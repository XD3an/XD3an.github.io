檔案格式剖析：一切的一切從檔案格式（File Format）開始

Posted by XD3an on November 13, 2023

在電腦（Computer）系統中，檔案格式是一種用於標準化作業的的格式，為了讓所有不同的電腦可以依據相同的標準對不同類型的檔案進行處理，因此檔案格式的標準在電腦系統中相當重要。

以下為常見的檔案格式依據內容及用途進行區分：

  • 二進位檔案（可執行檔案）：這類檔案主要的用途是儲存程式碼（機器碼）與資料，可以直接在電腦上執行或連結，進而執行程式內容。
    • Windows
        • PE (Portable Executable)
    • Linux&Unix
        • ELF(Exectuable and Linked Format)
    • MacOS

  • 文字檔案：這類檔案主要用於儲存文字資訊，包含純文字及帶有一定格式的文字。
    • TXT
    • HTML
    • DOC
    • PDF
    • ...

  • 影像檔案：這類檔案主要儲存圖片資訊，可以是靜態或動態的圖像。
    • BMP
    • JPG/JPEG
      • 兩者區別僅有副檔名的長度，因為早期 Windows 與 DOS 系統只支援三個字母長度的副檔名，所以名稱被簡化為三個字母，其實本質上兩者是指同一種檔案格式。
    • PNG
    • GIF
    • TIFF
    • ...

  • 音訊檔案：這類檔案主要是用於儲存聲音資訊，可以是無失真或有失真的壓縮方式。
    • WAV
    • MP3
    • ...

  • 視訊檔案：這類檔案主要儲存影片資訊，包含無失真或有失真的壓縮方式。
    • MP4
    • AVI
    • ...

  • 壓縮檔案：這類檔案主要儲存其他檔案的壓縮版本，可以減少檔案的大小和數量。
    • ZIP
    • RAR
    • 7Z
    • ...
